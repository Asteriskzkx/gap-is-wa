ตารางที่ ก.1.21  คำอธิบายยูสเคสการลบบัญชีผู้ใช้ในระบบ

| หัวข้อ | รายละเอียด |
|---|---|
| รหัสยูสเคส | 21 |
| ชื่อยูสเคส | ลบบัญชีผู้ใช้ในระบบ |
| ผู้เกี่ยวข้อง | ผู้ดูแลระบบ |
| ยูสเคสอ้างอิง | ระบบสารสนเทศสำหรับการจัดการข้อมูลทางการเกษตรผลผลิตยางพาราตามมาตรฐานจีเอพี |
| กระบวนการ | 1. ผู้ดูแลระบบเลือกเมนูจัดการผู้ใช้ในระบบ<br>2. ผู้ดูแลระบบค้นหาและเลือกรายการผู้ใช้ที่ต้องการลบ<br>3. ระบบแสดงกล่องยืนยันการลบผู้ใช้<br>4. ผู้ดูแลระบบยืนยันการลบบัญชีผู้ใช้<br>5. ระบบลบบัญชีผู้ใช้ตามกติกาของระบบ<br>6. ระบบบันทึกเหตุการณ์การลบผู้ใช้<br>7. ระบบแสดงผลการลบและปรับปรุงรายการผู้ใช้ล่าสุด |
| กระบวนการ (ทางเลือก) | 1. หากผู้ดูแลระบบยกเลิกการยืนยัน ระบบไม่ลบข้อมูลผู้ใช้<br>2. หากไม่สามารถลบผู้ใช้ได้จากเงื่อนไขข้อมูลที่เกี่ยวข้อง ระบบแจ้งข้อผิดพลาด |
| เงื่อนไขก่อน | 1. ผู้ดูแลระบบเข้าสู่ระบบเรียบร้อยแล้ว<br>2. มีผู้ใช้เป้าหมายที่อยู่ในขอบเขตการลบของระบบ |
| เงื่อนไขหลัง | บัญชีผู้ใช้เป้าหมายถูกลบหรือถูกปิดการใช้งานตามนโยบายระบบเรียบร้อยแล้ว |
