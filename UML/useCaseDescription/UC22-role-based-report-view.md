ตารางที่ ก.1.22  คำอธิบายยูสเคสการเรียกดูรายงาน

| หัวข้อ | รายละเอียด |
|---|---|
| รหัสยูสเคส | 22 |
| ชื่อยูสเคส | การเรียกดูรายงาน |
| ผู้เกี่ยวข้อง | ผู้ตรวจประเมิน, คณะกรรมการ, ผู้ดูแลระบบ |
| ยูสเคสอ้างอิง | ระบบสารสนเทศสำหรับการจัดการข้อมูลทางการเกษตรผลผลิตยางพาราตามมาตรฐานจีเอพี |
| กระบวนการ | 1. ผู้ใช้เลือกเมนูตรวจสอบรายงานตามบทบาทของตน<br>2. ผู้ใช้กำหนดช่วงวันที่หรือเงื่อนไขกรองรายงาน<br>3. ระบบประมวลผลและแสดงผลรายงานสรุป<br>4. ผู้ใช้เลือกดูรายละเอียดรายงานในส่วนที่สนใจ<br>5. ระบบแสดงข้อมูลเชิงลึกในรูปแบบตารางหรือกราฟ<br>6. ผู้ใช้เลือกส่งออกรายงานตามสิทธิ์ที่ได้รับ<br>7. ระบบจัดเตรียมไฟล์รายงานและให้ผู้ใช้ดาวน์โหลด |
| กระบวนการ (ทางเลือก) | 1. หากไม่พบข้อมูลในเงื่อนไขที่เลือก ระบบแสดงรายงานว่าง<br>2. หากผู้ใช้เลือกส่งออกข้อมูลที่เกินสิทธิ์บทบาท ระบบแจ้งเตือนและปฏิเสธการดำเนินการ |
| เงื่อนไขก่อน | 1. ผู้ใช้เข้าสู่ระบบในบทบาทที่รองรับการดูรายงานแล้ว<br>2. ผู้ใช้สามารถเข้าถึงเมนูรายงานตามสิทธิ์บทบาทของตน |
| เงื่อนไขหลัง | ผู้ใช้ได้รับข้อมูลรายงานที่ต้องการบนหน้าจอหรือในรูปแบบไฟล์ส่งออก |
