ตารางที่ ก.1.13  คำอธิบายยูสเคสการบันทึกข้อมูลประจำสวนยาง

| หัวข้อ | รายละเอียด |
|---|---|
| รหัสยูสเคส | 13 |
| ชื่อยูสเคส | บันทึกข้อมูลประจำสวนยาง |
| ผู้เกี่ยวข้อง | ผู้ตรวจประเมิน |
| ยูสเคสอ้างอิง | ระบบสารสนเทศสำหรับการจัดการข้อมูลทางการเกษตรผลผลิตยางพาราตามมาตรฐานจีเอพี |
| กระบวนการ | 1. ผู้ตรวจประเมินเลือกเมนูบันทึกข้อมูลประจำสวนยาง<br>2. ผู้ตรวจประเมินเลือกรายการตรวจที่ต้องการบันทึกข้อมูล<br>3. ระบบโหลดข้อมูลเดิม (ถ้ามี) หรือแสดงฟอร์มใหม่<br>4. ผู้ตรวจประเมินระบุตำแหน่งแปลงและกรอกข้อมูลสภาพแปลง<br>5. ผู้ตรวจประเมินกรอกข้อมูลระบบน้ำ ปุ๋ย พืชร่วม และข้อมูลโรคพืช<br>6. ผู้ตรวจประเมินตรวจทานข้อมูลที่บันทึก<br>7. ผู้ตรวจประเมินยืนยันการบันทึกข้อมูล<br>8. ระบบสร้างหรืออัปเดตข้อมูลประจำสวนยางสำเร็จ |
| กระบวนการ (ทางเลือก) | 1. หากยังไม่เลือกรายการตรวจ ระบบไม่อนุญาตให้บันทึกข้อมูล<br>2. หากข้อมูลจำเป็นไม่ครบ ระบบแจ้งให้กรอกข้อมูลเพิ่มเติมก่อนบันทึก |
| เงื่อนไขก่อน | 1. ผู้ตรวจประเมินเข้าสู่ระบบเรียบร้อยแล้ว<br>2. มีรายการตรวจที่ผู้ตรวจประเมินรับผิดชอบอยู่ในระบบ |
| เงื่อนไขหลัง | ระบบมีข้อมูลประจำสวนยางที่เป็นปัจจุบันสำหรับรายการตรวจที่เลือก |
