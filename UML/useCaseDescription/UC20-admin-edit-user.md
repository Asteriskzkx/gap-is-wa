ตารางที่ ก.1.20  คำอธิบายยูสเคสการแก้ไขข้อมูลผู้ใช้ในระบบ

| หัวข้อ | รายละเอียด |
|---|---|
| รหัสยูสเคส | 20 |
| ชื่อยูสเคส | แก้ไขข้อมูลผู้ใช้ในระบบ |
| ผู้เกี่ยวข้อง | ผู้ดูแลระบบ |
| ยูสเคสอ้างอิง | ระบบสารสนเทศสำหรับการจัดการข้อมูลทางการเกษตรผลผลิตยางพาราตามมาตรฐานจีเอพี |
| กระบวนการ | 1. ผู้ดูแลระบบเลือกเมนูจัดการผู้ใช้ในระบบ<br>2. ผู้ดูแลระบบค้นหาและเลือกรายการผู้ใช้ที่ต้องการแก้ไข<br>3. ระบบแสดงข้อมูลผู้ใช้ปัจจุบัน<br>4. ผู้ดูแลระบบปรับปรุงข้อมูลผู้ใช้ที่อนุญาตให้แก้ไข<br>5. ผู้ดูแลระบบยืนยันการแก้ไขข้อมูล<br>6. ระบบบันทึกข้อมูลผู้ใช้ใหม่และบันทึกเหตุการณ์การแก้ไข<br>7. ระบบแสดงผลการแก้ไขสำเร็จแก่ผู้ดูแลระบบ |
| กระบวนการ (ทางเลือก) | 1. หากข้อมูลที่แก้ไขไม่ผ่านเงื่อนไข ระบบแจ้งให้แก้ไขก่อนบันทึก<br>2. หากไม่พบผู้ใช้เป้าหมาย ระบบแจ้งข้อผิดพลาดและไม่ดำเนินการต่อ |
| เงื่อนไขก่อน | 1. ผู้ดูแลระบบเข้าสู่ระบบเรียบร้อยแล้ว<br>2. มีข้อมูลผู้ใช้ในระบบที่สามารถเลือกเพื่อแก้ไขได้ |
| เงื่อนไขหลัง | ข้อมูลผู้ใช้ที่เลือกถูกอัปเดตล่าสุดในระบบเรียบร้อยแล้ว |
