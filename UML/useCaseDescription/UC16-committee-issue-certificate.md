ตารางที่ ก.1.16  คำอธิบายยูสเคสการออกใบรับรองแหล่งผลิต GAP ยางพารา

| หัวข้อ | รายละเอียด |
|---|---|
| รหัสยูสเคส | 16 |
| ชื่อยูสเคส | ออกใบรับรองแหล่งผลิต GAP ยางพารา |
| ผู้เกี่ยวข้อง | คณะกรรมการ |
| ยูสเคสอ้างอิง | ระบบสารสนเทศสำหรับการจัดการข้อมูลทางการเกษตรผลผลิตยางพาราตามมาตรฐานจีเอพี |
| กระบวนการ | 1. คณะกรรมการเลือกเมนูออกใบรับรองแหล่งผลิต GAP ยางพารา<br>2. ระบบแสดงรายการตรวจที่ผ่านและพร้อมออกใบรับรอง<br>3. คณะกรรมการเลือกรายการตรวจที่ต้องการออกใบรับรอง<br>4. คณะกรรมการกำหนดวันที่มีผลและวันที่หมดอายุของใบรับรอง<br>5. คณะกรรมการอัปโหลดไฟล์ใบรับรอง<br>6. คณะกรรมการยืนยันการออกใบรับรอง<br>7. ระบบบันทึกข้อมูลใบรับรองและเชื่อมโยงไฟล์เอกสาร<br>8. ระบบแสดงผลการออกใบรับรองสำเร็จ |
| กระบวนการ (ทางเลือก) | 1. หากรายการตรวจไม่เข้าเงื่อนไขการออกใบรับรอง ระบบไม่อนุญาตให้ดำเนินการ<br>2. หากข้อมูลวันที่หรือไฟล์เอกสารไม่ถูกต้อง ระบบแจ้งให้แก้ไขก่อนยืนยัน |
| เงื่อนไขก่อน | 1. คณะกรรมการเข้าสู่ระบบเรียบร้อยแล้ว<br>2. มีรายการตรวจที่ผ่านเกณฑ์และยังไม่ถูกออกใบรับรอง |
| เงื่อนไขหลัง | ระบบมีใบรับรองแหล่งผลิตใหม่ในสถานะใช้งาน พร้อมข้อมูลวันที่และไฟล์ใบรับรอง |
