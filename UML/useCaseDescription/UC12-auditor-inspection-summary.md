ตารางที่ ก.1.12  คำอธิบายยูสเคสการสรุปผลการตรวจประเมิน

| หัวข้อ | รายละเอียด |
|---|---|
| รหัสยูสเคส | 12 |
| ชื่อยูสเคส | สรุปผลการตรวจประเมิน |
| ผู้เกี่ยวข้อง | ผู้ตรวจประเมิน |
| ยูสเคสอ้างอิง | ระบบสารสนเทศสำหรับการจัดการข้อมูลทางการเกษตรผลผลิตยางพาราตามมาตรฐานจีเอพี |
| กระบวนการ | 1. ผู้ตรวจประเมินเลือกเมนูสรุปผลการตรวจประเมิน<br>2. ระบบแสดงรายการตรวจที่เกี่ยวข้องตามสถานะ<br>3. ผู้ตรวจประเมินเลือกรายการตรวจที่ต้องการดูสรุป<br>4. ระบบแสดงผลรวมของรายการตรวจและผลผ่าน/ไม่ผ่าน<br>5. ผู้ตรวจประเมินเปิดดูรายละเอียดรายหัวข้อเพิ่มเติมได้<br>6. ผู้ตรวจประเมินใช้ข้อมูลสรุปเพื่อยืนยันความครบถ้วนของรายงาน |
| กระบวนการ (ทางเลือก) | 1. หากไม่พบรายการตรวจตามเงื่อนไข ระบบแสดงรายการว่าง<br>2. หากพบข้อมูลบางหัวข้อยังไม่สมบูรณ์ ระบบแสดงเตือนและให้กลับไปบันทึกข้อมูลเพิ่มเติม |
| เงื่อนไขก่อน | 1. ผู้ตรวจประเมินเข้าสู่ระบบเรียบร้อยแล้ว<br>2. มีรายการตรวจที่เสร็จสิ้นหรืออยู่ในขั้นตอนสรุปผลในระบบ |
| เงื่อนไขหลัง | ผู้ตรวจประเมินรับทราบผลสรุปการตรวจและข้อมูลพร้อมสำหรับการพิจารณาขั้นต่อไป |
