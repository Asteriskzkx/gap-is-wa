ตารางที่ ก.1.1  คำอธิบายยูสเคสการสมัครสมาชิก

| หัวข้อ | รายละเอียด |
|---|---|
| รหัสยูสเคส | 1 |
| ชื่อยูสเคส | การสมัครสมาชิก |
| ผู้เกี่ยวข้อง | เกษตรกร |
| ยูสเคสอ้างอิง | ระบบสารสนเทศสำหรับการจัดการข้อมูลทางการเกษตรผลผลิตยางพาราตามมาตรฐานจีเอพี |
| กระบวนการ | 1. ผู้ใช้เข้าสู่หน้าเข้าสู่ระบบของระบบ<br>2. ผู้ใช้เลือกเมนูสมัครสมาชิก<br>3. ผู้ใช้กรอกข้อมูลบัญชีผู้ใช้ (อีเมลและรหัสผ่าน)<br>4. ผู้ใช้กรอกข้อมูลส่วนตัวและข้อมูลที่อยู่<br>5. ผู้ใช้กรอกข้อมูลติดต่อและยอมรับเงื่อนไขการใช้งาน<br>6. ผู้ใช้กดยืนยันการสมัครสมาชิก<br>7. ระบบตรวจสอบความถูกต้องของข้อมูลและบันทึกบัญชีใหม่<br>8. ระบบแสดงผลการสมัครสมาชิกสำเร็จ |
| กระบวนการ (ทางเลือก) | 1. หากข้อมูลไม่ครบหรือรูปแบบไม่ถูกต้อง ระบบแจ้งให้แก้ไขก่อนยืนยัน<br>2. หากอีเมลซ้ำกับบัญชีเดิม ระบบปฏิเสธการสมัครและแจ้งผู้ใช้ |
| เงื่อนไขก่อน | 1. ผู้ใช้สามารถเข้าถึงระบบผ่านเว็บแอปพลิเคชันได้<br>2. ผู้ใช้ยังไม่มีบัญชีผู้ใช้ในระบบ |
| เงื่อนไขหลัง | เกษตรกรมีบัญชีผู้ใช้ในระบบและสามารถเข้าสู่ระบบได้ |
