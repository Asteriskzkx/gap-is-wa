ตารางที่ ก.1.4  คำอธิบายยูสเคสการติดตามสถานะการยื่นขอใบรับรอง

| หัวข้อ | รายละเอียด |
|---|---|
| รหัสยูสเคส | 4 |
| ชื่อยูสเคส | ติดตามสถานะการยื่นขอใบรับรอง |
| ผู้เกี่ยวข้อง | เกษตรกร |
| ยูสเคสอ้างอิง | ระบบสารสนเทศสำหรับการจัดการข้อมูลทางการเกษตรผลผลิตยางพาราตามมาตรฐานจีเอพี |
| กระบวนการ | 1. ผู้ใช้เลือกเมนูติดตามสถานะการรับรอง<br>2. ระบบแสดงรายการคำขอรับรองทั้งหมดของผู้ใช้<br>3. ผู้ใช้ค้นหา กรอง หรือเรียงลำดับรายการคำขอ<br>4. ผู้ใช้เลือกคำขอที่ต้องการดูรายละเอียด<br>5. ระบบแสดงสถานะคำขอ เช่น รอการตรวจ กำหนดตรวจ ตรวจเสร็จแล้ว<br>6. ระบบแสดงข้อมูลประกอบที่เกี่ยวข้องกับคำขอนั้น |
| กระบวนการ (ทางเลือก) | 1. หากไม่พบคำขอตามเงื่อนไขที่เลือก ระบบแสดงผลลัพธ์ว่าง<br>2. หากคำขออยู่ในสถานะไม่ผ่านหรือยกเลิก ระบบแสดงสถานะล่าสุดและรายละเอียดที่เกี่ยวข้อง |
| เงื่อนไขก่อน | 1. ผู้ใช้เข้าสู่ระบบในบทบาทเกษตรกรเรียบร้อยแล้ว<br>2. ผู้ใช้มีคำขอรับรองหรือประวัติรายการในระบบ |
| เงื่อนไขหลัง | ผู้ใช้รับทราบสถานะล่าสุดของคำขอรับรองและสามารถวางแผนการดำเนินการต่อได้ |
