ตารางที่ ก.1.3  คำอธิบายยูสเคสการยื่นขอใบรับรองแหล่งผลิต GAP ยางพารา

| หัวข้อ | รายละเอียด |
|---|---|
| รหัสยูสเคส | 3 |
| ชื่อยูสเคส | ยื่นขอใบรับรองแหล่งผลิต GAP ยางพารา |
| ผู้เกี่ยวข้อง | เกษตรกร |
| ยูสเคสอ้างอิง | ระบบสารสนเทศสำหรับการจัดการข้อมูลทางการเกษตรผลผลิตยางพาราตามมาตรฐานจีเอพี |
| กระบวนการ | 1. ผู้ใช้เลือกเมนูยื่นขอใบรับรองแหล่งผลิต<br>2. ผู้ใช้กรอกข้อมูลแปลงสวนยางและตำแหน่งที่ตั้ง<br>3. ผู้ใช้กรอกรายละเอียดการปลูกยางพารา<br>4. ผู้ใช้ตรวจสอบข้อมูลสรุปก่อนส่งคำขอ<br>5. ผู้ใช้ยืนยันความถูกต้องของข้อมูล<br>6. ผู้ใช้กดส่งคำขอรับรอง<br>7. ระบบบันทึกคำขอและกำหนดสถานะเริ่มต้นเพื่อรอการตรวจประเมิน |
| กระบวนการ (ทางเลือก) | 1. หากข้อมูลแปลงหรือรายละเอียดการปลูกไม่ครบ ระบบแจ้งให้แก้ไขก่อนส่งคำขอ<br>2. หากผู้ใช้ยกเลิกก่อนยืนยัน ระบบไม่บันทึกคำขอใหม่ |
| เงื่อนไขก่อน | 1. ผู้ใช้เข้าสู่ระบบในบทบาทเกษตรกรเรียบร้อยแล้ว<br>2. ผู้ใช้มีสิทธิ์ยื่นคำขอรับรองในระบบ |
| เงื่อนไขหลัง | ระบบมีข้อมูลคำขอรับรองแหล่งผลิตใหม่ในสถานะรอการตรวจประเมิน |
