ตารางที่ ก.1.7  คำอธิบายยูสเคสการเข้าสู่ระบบสำหรับผู้ตรวจประเมิน

| หัวข้อ | รายละเอียด |
|---|---|
| รหัสยูสเคส | 7 |
| ชื่อยูสเคส | การเข้าสู่ระบบ (ผู้ตรวจประเมิน) |
| ผู้เกี่ยวข้อง | ผู้ตรวจประเมิน |
| ยูสเคสอ้างอิง | ระบบสารสนเทศสำหรับการจัดการข้อมูลทางการเกษตรผลผลิตยางพาราตามมาตรฐานจีเอพี |
| กระบวนการ | 1. ผู้ตรวจประเมินเข้าสู่หน้าเข้าสู่ระบบ<br>2. ผู้ตรวจประเมินเลือกบทบาทผู้ตรวจประเมิน<br>3. ผู้ตรวจประเมินกรอกอีเมลและรหัสผ่าน<br>4. ผู้ตรวจประเมินกดปุ่มเข้าสู่ระบบ<br>5. ระบบตรวจสอบข้อมูลบัญชีและสิทธิ์บทบาท<br>6. ระบบเข้าสู่ระบบสำเร็จและนำไปยังหน้าหลักผู้ตรวจประเมิน |
| กระบวนการ (ทางเลือก) | 1. หากข้อมูลบัญชีไม่ถูกต้อง ระบบแจ้งข้อผิดพลาดและไม่อนุญาตให้เข้าใช้งาน<br>2. หากบทบาทไม่ตรงกับสิทธิ์ของบัญชี ระบบปฏิเสธการเข้าสู่ระบบ |
| เงื่อนไขก่อน | 1. ผู้ตรวจประเมินสามารถเข้าถึงหน้าล็อกอินของระบบได้<br>2. ผู้ตรวจประเมินมีบัญชีผู้ใช้ในระบบแล้ว |
| เงื่อนไขหลัง | ผู้ตรวจประเมินเข้าสู่ระบบสำเร็จและพร้อมใช้งานฟังก์ชันในบทบาทผู้ตรวจประเมิน |
