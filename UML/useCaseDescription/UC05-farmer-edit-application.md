ตารางที่ ก.1.5  คำอธิบายยูสเคสการแก้ไขข้อมูลคำขอใบรับรอง

| หัวข้อ | รายละเอียด |
|---|---|
| รหัสยูสเคส | 5 |
| ชื่อยูสเคส | แก้ไขข้อมูลคำขอใบรับรอง |
| ผู้เกี่ยวข้อง | เกษตรกร |
| ยูสเคสอ้างอิง | ระบบสารสนเทศสำหรับการจัดการข้อมูลทางการเกษตรผลผลิตยางพาราตามมาตรฐานจีเอพี |
| กระบวนการ | 1. ผู้ใช้เลือกเมนูแก้ไขข้อมูลคำขอใบรับรอง<br>2. ระบบแสดงรายการคำขอหรือแปลงที่สามารถแก้ไขได้<br>3. ผู้ใช้เลือกรายการที่ต้องการแก้ไข<br>4. ระบบแสดงข้อมูลเดิมเพื่อให้แก้ไข<br>5. ผู้ใช้ปรับปรุงข้อมูลแปลงและรายละเอียดการปลูก<br>6. ผู้ใช้ยืนยันการแก้ไข<br>7. ระบบบันทึกข้อมูลใหม่และแสดงผลการแก้ไขสำเร็จ |
| กระบวนการ (ทางเลือก) | 1. หากข้อมูลที่แก้ไขไม่ผ่านเงื่อนไข ระบบแจ้งให้แก้ไขก่อนบันทึก<br>2. หากรายการนั้นไม่อยู่ในสถานะที่อนุญาตให้แก้ไข ระบบปฏิเสธการแก้ไข |
| เงื่อนไขก่อน | 1. ผู้ใช้เข้าสู่ระบบในบทบาทเกษตรกรเรียบร้อยแล้ว<br>2. มีคำขอหรือข้อมูลที่อยู่ในสถานะแก้ไขได้ในระบบ |
| เงื่อนไขหลัง | ข้อมูลคำขอใบรับรองของผู้ใช้ถูกปรับปรุงล่าสุดในระบบ |
