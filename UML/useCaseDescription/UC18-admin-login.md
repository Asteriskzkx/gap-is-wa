ตารางที่ ก.1.18  คำอธิบายยูสเคสการเข้าสู่ระบบสำหรับผู้ดูแลระบบ

| หัวข้อ | รายละเอียด |
|---|---|
| รหัสยูสเคส | 18 |
| ชื่อยูสเคส | การเข้าสู่ระบบ (ผู้ดูแลระบบ) |
| ผู้เกี่ยวข้อง | ผู้ดูแลระบบ |
| ยูสเคสอ้างอิง | ระบบสารสนเทศสำหรับการจัดการข้อมูลทางการเกษตรผลผลิตยางพาราตามมาตรฐานจีเอพี |
| กระบวนการ | 1. ผู้ดูแลระบบเข้าสู่หน้าเข้าสู่ระบบ<br>2. ผู้ดูแลระบบเลือกบทบาทผู้ดูแลระบบ<br>3. ผู้ดูแลระบบกรอกอีเมลและรหัสผ่าน<br>4. ผู้ดูแลระบบกดปุ่มเข้าสู่ระบบ<br>5. ระบบตรวจสอบข้อมูลบัญชีและบทบาทผู้ดูแลระบบ<br>6. ระบบนำผู้ใช้เข้าสู่หน้าหลักผู้ดูแลระบบ |
| กระบวนการ (ทางเลือก) | 1. หากข้อมูลเข้าสู่ระบบไม่ถูกต้อง ระบบแจ้งข้อผิดพลาดและไม่อนุญาตให้เข้าใช้งาน<br>2. หากบัญชีไม่มีสิทธิ์ผู้ดูแลระบบ ระบบปฏิเสธการเข้าสู่ระบบ |
| เงื่อนไขก่อน | 1. ผู้ดูแลระบบสามารถเข้าถึงหน้าล็อกอินของระบบได้<br>2. ผู้ดูแลระบบมีบัญชีในระบบพร้อมสิทธิ์ผู้ดูแลระบบ |
| เงื่อนไขหลัง | ผู้ดูแลระบบเข้าสู่ระบบสำเร็จและพร้อมใช้งานฟังก์ชันบริหารจัดการระบบ |
